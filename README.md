# practicinglogicmath

A simple **calculator** built using **HTML, CSS, and JavaScript**. This calculator performs basic arithmetic operations with a user-friendly interface.

## ✨ Features

- ✅ **Basic Operations**: Addition (+), Subtraction (-), Multiplication (*), Division (/)
- ✅ **Clear (C) Button** – Resets the calculator
- ✅ **Backspace Button** – Deletes the last entered digit
- ✅ **Real-time Display Updates**
- ✅ **Handles Decimal Points** properly

## 🔧 Tech Stack

- **HTML** – Page structure
- **CSS** – Styling and responsiveness
- **JavaScript** – Handling user interactions and calculations

## 🧠 Skills & Logic Learned

### **JavaScript Concepts**
- ✅ **DOM Manipulation** – Updating the calculator display dynamically
- ✅ **Event Listeners** – Handling button clicks efficiently
- ✅ **String Operations** – Managing user inputs before parsing
- ✅ **Conditional Statements** – Handling different operations and edge cases
- ✅ **Functions** – Reusable functions for calculations
